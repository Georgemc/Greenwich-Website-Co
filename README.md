# Greenwich Web Co.

Marketing and lead-funnel website for Greenwich Web Co. — a web-design studio
building modern websites for local businesses across the Greenwich, CT area.

## What this is
A single self-contained page (`index.html`) with no build step and no
dependencies beyond Google Fonts. Open the file in a browser, or serve the folder.

## Edit it
See [EDITING-GUIDE.md](EDITING-GUIDE.md) for how to change text, photos, and colors safely.

## Run locally
Just open `index.html` in a browser, or serve the folder:

```bash
npx serve .
```

## Deploy
Hosted on Vercel, connected to this GitHub repo. Pushing to the `main` branch
deploys automatically.
